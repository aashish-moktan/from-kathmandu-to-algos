# From Kathmandu to Algos 🚀

A personal journey from fundamentals to advanced algorithms and problem solving.

This repository is not just a collection of solutions — it's a structured attempt to deeply understand **data structures, algorithms, and problem-solving patterns**.

---

## 🎯 Goal

- Build strong intuition for algorithms
- Move beyond "just solving" problems → understand _why_ solutions work
- Recognize patterns across problems
- Prepare for real-world system thinking (not just interviews)

---

## 📚 What You'll Find Here

This repo will gradually include:

### 🧩 Data Structures

- Arrays
- Strings
- HashMaps / Sets
- Linked Lists
- Stacks & Queues
- Trees & Binary Trees
- Heaps / Priority Queues
- Graphs

---

### ⚙️ Algorithms & Patterns

- Two Pointers
- Sliding Window
- Recursion & Backtracking
- Divide & Conquer
- Dynamic Programming
- Greedy Algorithms
- Graph Traversal (BFS / DFS)
- Sorting & Searching

---

### 🧠 Problem Solving Approach

Each problem (eventually) will include:

- ✅ Problem statement (or link)
- 💡 Intuition
- 🧪 Dry run / example
- ⚙️ Code implementation
- ⏱ Time & Space complexity
- 🔁 Pattern recognition (where else this applies)
